# goit-algo-hw-07 - Trees & Balancing

## Task 1 - Max in BST/AVL
Функція `find_max(root)` знаходить найбільше значення.
Логіка: у BST/AVL найбільший елемент знаходиться у крайньому правому вузлі, тому рухаємося `right` поки можливо.

## Task 2 - Min in BST/AVL
Функція `find_min(root)` знаходить найменше значення.
Логіка: у BST/AVL найменший елемент знаходиться у крайньому лівому вузлі, тому рухаємося `left` поки можливо.

## Task 3 - Sum of all values
Функція `sum_values(root)` рахує суму всіх значень у дереві.
Реалізація: ітеративний обхід зі стеком (DFS), щоб не залежати від глибини рекурсії.
